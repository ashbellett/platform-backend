# platform-backend
A backend capable of CRUD operations using GraphQL, Apollo and MongoDB.

## Dependencies
- [Node.js](https://nodejs.org/)
- [MongoDB](https://www.mongodb.com/)

## Instructions
Install dependencies.
```
npm i
```
Lint code.
```
npm run lint
```
Run in development mode.
```
npm run dev
```
Transpile Typescript to Javascript.
```
npm run build
```
Run built app.
```
npm run start
```
